HORSE STABLE NAME FALLBACK PATCH

Upload the CONTENTS of this folder to GitHub.

Commit message:
Use stable name as horse fallback

This patch changes horse selection dropdowns:
- Work, vet/treatments, farrier, feed, gear, race records, invoices and updates now show the horse's Registered Name if entered.
- If Registered Name is blank, they show/use Stable Name instead.
- This allows young/unregistered horses to still be selectable throughout the app.
