MEDIA UPLOAD PATCH

Upload the CONTENTS of this folder to GitHub.

Then run RUN_THIS_STORAGE_SQL_FOR_MEDIA_UPLOADS.sql in Supabase SQL Editor.

This adds:
- direct photo uploads in Updates
- direct video uploads in Updates
- uploaded files stored in Supabase Storage bucket: update-media
- URLs automatically saved into the update record
- old manual URL paste support still works

Vercel settings remain:
Framework: Vite
Install Command: pnpm install --no-frozen-lockfile OR npm install --no-audit --no-fund
Build Command: pnpm run build OR npm run build
Output Directory: dist
